# Assignment

Translate this JavaScript code into Python.

```javascript
function sign(n)
{
    if ( n > 0 )
    {
        return 1;
    }
    else if ( n < 0 )
    {
        return -1;
    }
    else
    {
        return 0;
    }
}
```
